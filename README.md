 Mini Project: Student Manager

This mini project demonstrates how to use Java Collections and Map to manage students.

 Features
- Add students
- Remove students by id
- Print all students
- Filter top students by grade using Stream API

 Classes

 Student
Represents a student with:
- id
- name
- grade

 StudentManager
Uses `HashMap<Integer, Student>` to:
- store students
- search by id
- filter by grade

 Main
Runs the demo:
- adds students
- prints all
- shows top students
- removes one student

 Technologies
- Java
- Collections (Map, List)
- Stream API (filter)
- Lambda expressions
